Class: BackgroundLayer

BackgroundLayer(props)

new BackgroundLayer(props)

Background Graphics layer.
Intended for backgrounds that don't change much; sprites are ok, but moving elements will require marking the entire layer dirty for proper functionality. Generally, the use of Screen.createLayer("BackgroundLayer") is preferred over creating layer by hand.
Parameters:
Name Type Description
props Object The properties to create this layer with.
Properties
Name Type Description
screenContext Screen The parent screen for this layer.
canvasContextWrapper CanvasContextWrapper The canvasContextWrapper. This layer will draw to the canvas' context, via wrapper's exposed methods.
width number The width of the layer. Should match Screen.
height number The height of the layer. Should match Screen.
