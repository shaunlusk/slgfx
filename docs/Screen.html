Class: Screen

Screen(props)

new Screen(props)

The Screen is the overriding container for Graphics components. The Screen orchestrates updating and rendering its layers, propagates mouse events down to the layers, and notifies event listeners when events occur.
Parameters:
Name Type Description
props Object Properties

 Optional. An HTMLElement to write Frames-per-second information to.
