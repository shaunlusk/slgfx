Class MoveOrder

Directs movement of {@link GfxElements} when moveTo() is called. These are created and handled internally in GfxElements.

param element

The element this MoveOrder will be bound to.

param tx

The target x location for the element.

param ty

The target y location for the element.

param duration

The length of time (milliseconds) for the movement to the target location.

param elementCallback

The callback of the parent element; called when movement is done.

param [callback]

Optional callback to be called when movement is done.
