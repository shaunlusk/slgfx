Class Layer

Abstract class for graphical layers on a Screen.
Existing implementations: BackgroundLayer, GfxLayer

param props

Configuration properties.

param props.screenContext

The parent screen.

param props.canvasContextWrapper

The canvasContextWrapper. This layer will draw to the canvas' context, via wrapper's exposed methods.

param props.width

The width of the layer. Should match Screen.

param props.height

The height of the layer. Should match Screen.
