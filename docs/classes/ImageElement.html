Class ImageElement

Graphics element that renders part or all of an Image.
It is good practice to have a single or few Images that have many tiles on them to make efficient use of memory and screen drawing. Using ImageElement, you can treat tiles from the source image as discrete screen elements that can be moved and interacted with. For animations, see ImageSprite

param props

The properties for this element.

param props.screenContext

The target screen.

param [props.scaleX=1]

Horizontal scale of this element. Independent of screen scale.

param [props.scaleY=1]

Vertical scale of this element. Independent of screen scale.

param [props.hidden=false]

Whether to hide this element.

param [props.x=0]

The X coordinate for this element.

param [props.y=0]

The Y coordinate for this element.

param [props.rotation=0]

The amount of rotation to apply to the element, in radians. Applied on top of base rotation.

param [props.baseRotation=0]

The amount of base rotation to apply to the element, in radians. Usually used to apply an initial, unchanging rotation to the element. Useful for correcting orientation of images.

param [props.horizontalFlip=false]

Whether to flip the element horizontally.

param [props.verticalFlip=false]

Whether to flip the element vertically.

param [props.zIndex=-1]

The z-index; elements with higher zIndex values will be drawn later than those with lower values (drawn on top of those with lower values).

param props.image

The image to use for this element. Can be created via html (<img>) or javascript (new Image()).

param props.sourceX

The x starting point of the desired subsection of the image

param props.sourceY

The y starting point of the desired subsection of the image

param props.sourceWidth

The width of the desired subsection of the image

param props.sourceHeight

The height of the desired subsection of the image

param props.width

The desired width of the ImageElement; if this differs from the source dimensions, the image will be stretched or shrunk accordingly

param props.height

The desired height of the ImageElement; if this differs from the source dimensions, the image will be stretched or shrunk accordingly

param props.imageRenderer

The ImageRenderer that will draw on the canvas.

  • Inherited from GfxElement.nudge

    Nudge the element, make it move back and forth rapidly. Provides options for decays of movement range (will make movement less extreme over time), and of time interval (movement becomes more rapid over time). Automatically returns to its starting position upon completion. If the offsets or the interval time reaches zero due to the decay parameters, the nudge will finish. Will throw an error if the parameters will cause the nudge to repeat for >= 1000 intervals.

    fires

    GfxElement#ELEMENT_STARTED_MOVING

    fires

    GfxElement#ELEMENT_STOPPED_MOVING

    Parameters

    • offsetX: number

      The maximum (positive and negative) horizontal offset to move the element.

    • offsetY: number

      The maximum (positive and negative) vertical offset to move the element.

    • decay: number

      After hitting the current offset and reversing direction, reduce the offsets by this amount.

    • interval: number

      How rapidly (milliseconds) the element should move to the offset, before reversing direction.

    • intervalDecay: number

      Reduces the interval time after each iteration.

    • Optional callback: () => void

      A function to call when the nudge has completed.

        • (): void

        • Returns void

    Returns void
