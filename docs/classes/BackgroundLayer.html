Class BackgroundLayer

Background Graphics layer.
Intended for backgrounds that don't change much; sprites are ok, but moving elements will require marking the entire layer dirty for proper functionality. Generally, the use of Screen.createLayer("BackgroundLayer") is preferred over creating layer by hand.

param props

The properties to create this layer with.

param props.screenContext

The parent screen for this layer.

param props.canvasContextWrapper

The canvasContextWrapper. This layer will draw to the canvas' context, via wrapper's exposed methods.

param props.width

The width of the layer. Should match Screen.

param props.height

The height of the layer. Should match Screen.
