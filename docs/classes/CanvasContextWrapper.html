Class CanvasContextWrapper

CanvasContextWrapper - wraps a CanvasRenderingContext2D to provide viewport capabilities.

param props
param props.width

The width; should match the width of the canvas element.

param props.height

The height; should match the height of the canvas element.

param props.canvasContext

The canvasContext element.

param [props.viewOriginX=0]

X Coordinate of the viewport

param [props.viewOriginY=0]

y Coordinate of the viewport

param [props.imageSmoothingEnabled=false]

Whether to use image smoothing.

see

https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D
