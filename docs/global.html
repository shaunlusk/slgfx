ElementMouseEvent

A Mouse Event on a GfxElement.
Type:
  • Object
Properties:
Name Type Description
type string EventType
data Object The data provided by the event emitter.
Properties
Name Type Description
x number The functional x coordinate of the mouse, incorporating the scale of canvas context wrapper view origin.
y number The functional y coordinate of the mouse, incorporating the scale of canvas context wrapper view origin.
viewOriginAdjustedX number The x coordinate of the mouse, relative to the view origin of the canvas context wrapper.
viewOriginAdjustedY number The y coordinate of the mouse, relative to the view origin of the canvas context wrapper.

element GfxElement The element that the mouse interacted with.
